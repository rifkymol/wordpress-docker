## 5.4 Responsive Design

### Breakpoints Standard

| Device | Width | Prioritas |
|--------|-------|-----------|
| Mobile | < 768px | Tinggi |
| Tablet | 768-1024px | Sedang |
| Desktop | > 1024px | Tinggi |

### Testing Responsive

#### Browser DevTools:
1. Klik kanan > Inspect
2. Toggle device toolbar (Ctrl+Shift+M)
3. Test berbagai ukuran layar

#### Online Tools:
- Responsinator.com
- BrowserStack.com
- Google Mobile-Friendly Test

### Tips Responsive E-Commerce:
1. **Touch-friendly buttons**: Minimal 44x44 px
2. **Readable text**: Minimal 16px font size
3. **Simplified navigation**: Hamburger menu
4. **Fast loading**: Optimize images
5. **Easy checkout**: Single column form
