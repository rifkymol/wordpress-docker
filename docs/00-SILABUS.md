# Silabus: Membangun Website E-Commerce dengan WordPress & Docker

## Tujuan Pembelajaran
Setelah menyelesaikan kursus ini, Anda akan mampu:
- Memahami konsep Docker dan containerization
- Setup dan konfigurasi WordPress profesional
- Membangun toko online dengan WooCommerce
- Mengoptimalkan performa dan keamanan website
- Deploy website ke production server

## Prasyarat
- Komputer dengan minimal 8GB RAM
- Windows 10/11 dengan Docker Desktop terinstall
- Koneksi internet stabil
- Text editor (VS Code direkomendasikan)
- Pengetahuan dasar HTML/CSS (opsional tapi membantu)

## Tools yang Digunakan
- Docker Desktop
- VS Code
- Browser (Chrome/Firefox dengan DevTools)
- Terminal/PowerShell

## Estimasi Total Waktu
**~80 jam** (10 minggu @ 8 jam/minggu)
